Computer Access

The computers at ASH Russellville Specialty Care and ASH Hot Springs Village (HSV) are members of the ASH domain.  If a clinic requires access to these desktop PCs, they must contact ASH Information Services to set up a user account (they will need to sign some documentation before given access). ASH Information systems can also work with a clinic’s IT provider to set up remote access to clinical systems (if this is possible).  We ask the clinics to have their IT provider contact ASH Information Systems in advance to discuss.

Bowen Hefley Ortho Remote Scan


BH Ortho uses the application Remote Scan by Quest to map the local scanners on their front desk computers in Russellville and Hot Spring Village.  This allows you to map USB scanners to Remote Desktop sessions that have the Remote Scan client installed. At this time it is primarily used for BH Ortho to connect to their RDS server ts.orthosurgeons.com.

Troubleshooting Help

Bowen Hefley Orthopedic Unable to Send Studies to their PACS Server

Problem – Users from BH Ortho Report issues sending studies from ASH Specialty Clinic (in Hot Springs or Russellville) to the BH Ortho PACS system

Troubleshooting

  • First find if the user is in Hot Springs Village or Russellville
  • Connect to the end user’s machine, and launch a CMD session
  • Attempt to pint the BH Ortho PACS Server: 192.168.230.228
  • If you cannot ping the server you will need to restart the ASH Clinic <> BH Ortho VPN Tunnel

Restarting the Site to Site VPN Tunnel

  • Sign in to the ASH Specialty Clinic Firewall via the web
    • ASH Russellville CheckPoint 750: https://edafio.itglue.com/3540406/passwords/17131755
    • ASH Hot Springs Village CheckPoint 750: https://edafio.itglue.com/3540406/passwords/17131756
  • In the admin page navigate to VPN > Site to Site > VPN Sites
    • Highlight the Bowen Hefley Tunnel (remote IP Is 12.14.28.210)
    • Click on Disable to stop the tunnel

  • After 10 seconds Highlight the tunnel again and click on Enable


Have the end user attempt to send studies again.
